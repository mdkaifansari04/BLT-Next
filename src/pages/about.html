About OWASP BLT

Making bug reporting accessible, rewarding, and collaborative for everyone.

What is BLT?

The Bug Logging Tool

OWASP BLT is an open-source, community-driven platform that makes reporting, tracking, and fixing software bugs and vulnerabilities easy and collaborative. We combine gamification, bug bounties, and a vibrant global community to create a better, more secure internet.

Our Mission

Empowering Security Research

To empower security researchers, developers, and organizations worldwide with the tools and incentives needed to identify and fix security vulnerabilities — making the internet a safer place for everyone.

Key Features

What makes BLT special

Gamification

Earn points, badges, and climb leaderboards. Every bug you report builds your reputation in the security community.

Bug Bounties

Participate in bug bounty programs and earn real rewards. Organizations can launch their own programs on BLT.

Community-Driven

Join a global community of security researchers, developers, and ethical hackers collaborating to improve security.

Open Source

Fully open source and transparent. Contribute to the project, customize it for your needs, and help improve the platform.

The OWASP Foundation

BLT is a project of the OWASP Foundation, a nonprofit organization dedicated to improving software security. OWASP operates as a community of security professionals who develop freely available tools, documentation, and standards to help organizations build more secure software.
